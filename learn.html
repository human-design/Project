Common Tech Terms: Glossary


Front End

the part of an application that the user interacts with directly.

Back End

the part of an application that the user doesn’t see e.g. database, processing.

Full Stack

Both front end and back end

User Experience (UX)

the usability of a website or application and the overall experience of the user.

User Interface (UI)

anything a user may interact with to use a digital product or service. UI design focuses on the visual/aesthetic aspect of a website or application.

Server

A server is a computer that provides data to other computers.

API (Application Programming Interface)

the part of the server that receives requests and sends responses.

Programming Language

a vocabulary and set of grammatical rules for instructing a computer to perform specific tasks. Each language has its own set of keywords and a special syntax for organising programme instructions. E.g. JavaScript, Python, Ruby, C

Markup Language

a computer language that annotates text so that the computer can manipulate that text.  A markup language consists of easily understood keywords, names, or tags that help format the overall view of a page and the data it contains. HTML (Hypertext Markup Language) is the standard language for web pages and web applications.
